## 2. 機能詳細

### F-01 ユーザー登録

- 入力: メールアドレス、パスワード（、表示名）
- バリデーション:
  - メールアドレス形式チェック
  - メールアドレスは一意
  - パスワードは8文字以上
- パスワードはハッシュ化して保存
- 登録後、ログイン画面へ遷移

### F-02 ログイン／ログアウト

- 入力: メールアドレス、パスワード
- 認証成功: ボード一覧画面へ遷移
- 認証失敗: エラーメッセージを表示
- ログアウト: セッション破棄後、ログイン画面へ遷移

### F-03 ボード作成

- 入力: ボード名（必須、1〜50文字）
- 作成者 = ログインユーザー
- 作成後、ボード一覧に即時反映

### F-04 ボード一覧表示

- ログインユーザーが所有するボードのみ表示
- 新規作成ボタンを画面上部に配置

### F-05 ボード編集・削除

- ボード名変更: インライン編集または編集ダイアログ
- ボード削除: 確認ダイアログ表示後、ボード・配下のリスト・カードをまとめて削除

### F-06 リスト作成

- ボード詳細画面から「+ リストを追加」で作成
- 入力: リスト名（必須）
- 並び順（position）は末尾に自動割り当て

### F-07 リスト編集・削除

- リスト名変更: リスト上部のタイトルから編集
- リスト削除: 確認ダイアログ後、リストと配下カードを削除
- 並び替え: ドラッグ&ドロップでリスト順を変更

### F-08 カード作成

- リスト下部の「+ カードを追加」から作成
- 入力: カードタイトル（必須）

### F-09 カード編集・削除

- タイトル変更: カードのタイトルから編集
- 削除: 確認ダイアログ後に削除

### F-11 カード詳細

- 説明文（テキスト）
- 期限日（date）: 設定後はカード一覧に日付バッジを表示
  - 期限切れ・当日・2日以内・完了で色分け表示
- ラベル（緑/黄/橙/赤/紫/青 の6色から複数選択可）
- 優先度（高/中/低 から1つ選択、カード一覧にバッジ表示）

### F-10 カード移動・並び替え

- 同じリスト内: ドラッグで上下に並び替え
- 別のリスト間: ドラッグして別リストへドロップ
- position（並び順）を更新
- ソートボタン（優先度順・期限順）でリスト内カードを一括並び替え
  - ソート後もドラッグ&ドロップで自由に再並び替え可能

## 3. ユースケース

> **注記**: UC-01〜03 はログイン・ボード選択（F-01〜F-06）を前提とするが、現時点ではそれらは未実装。
> 現在の実装では UC-01 の手順 3〜5（カード操作部分）のみ動作する。

### UC-01: タスクを登録し進捗を更新する

1. ユーザーがログインする（F-02）
2. 対象のボードを開く（F-04）
3. 「ToDo」リストにカード「資料作成」を追加する（F-08）
4. 作業開始時に、カードを「Doing」リストへドラッグ&ドロップで移動する（F-10）
5. 完了したら「Done」リストへ移動する（F-10）

### UC-02: 新しいプロジェクト用のボードを準備する

1. ユーザーがログインする（F-02）
2. ボード一覧画面で「新規作成」ボタンを押し、ボード名「個人プロジェクトA」で作成する（F-03）
3. ボード詳細画面でリスト「ToDo」「Doing」「Done」を順に作成する（F-06）
4. 各リストにカードを追加してタスク登録を開始する（F-08）

### UC-03: 不要になったボードを削除する

1. ユーザーがボード一覧画面を開く（F-04）
2. 対象ボードの削除ボタンを押す
3. 確認ダイアログで「削除」を選択し、ボードと配下データをまとめて削除する（F-05）
